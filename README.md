# Asem Dekor Organizasyon

Asem Dekor için hazırlanmış, mobil uyumlu premium organizasyon ve ekipman kiralama web sitesi.

## Özellikler

- Responsive masaüstü, tablet ve mobil tasarım
- Hizmet ve ekipman kiralama bölümleri
- Tamamlanan organizasyonlar galerisi
- Telefon ve WhatsApp bağlantıları
- İletişim formu ve konum alanı
- SEO ve sosyal medya paylaşım görseli

## Gereksinimler

- Node.js 22.13 veya daha yeni bir sürüm
- npm

## Kurulum

```bash
npm install
npm run dev
```

Tarayıcıda `http://localhost:3000` adresini açın.

## Üretim derlemesi

```bash
npm run build
npm run start
```

## Görselleri değiştirme

Geçici ekipman ve organizasyon görselleri `public/media` klasöründedir. Yeni görseller aynı dosya adlarıyla değiştirilebilir veya `app/globals.css` içindeki görsel yolları güncellenebilir.

## Teknolojiler

- React
- Next.js uyumlu uygulama yapısı
- TypeScript
- Vinext / Vite
- Tailwind CSS
